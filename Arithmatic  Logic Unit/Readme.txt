README — 32‑bit ALU (SystemVerilog)
32‑bit Arithmetic Logic Unit (ALU)
This project implements a fully combinational 32‑bit ALU in SystemVerilog, supporting arithmetic, logical, shift, and comparison operations. It includes zero, carry, and signed‑overflow flag generation, making it suitable for CPU datapaths, FPGA designs, and digital logic coursework.

 Features
32‑bit operands (a, b)

9 supported operations:

ADD, SUB

AND, OR, XOR

Shift Left Logical (SLL)

Shift Right Logical (SRL)

Shift Right Arithmetic (SRA)

Set‑Less‑Than (SLT)

Carry‑out detection for ADD/SUB

Signed overflow detection

Zero flag output

Fully combinational, synthesizable design
